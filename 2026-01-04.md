What kind of "tool" is this? What does it do? What does it operate on, how do you access it? What are the input and outputs. Is the out put the same thing as the input? A hammer transforms a nail, but the nail is still the same nail, so in a way, the input is the same thing as the output, only in different position. 

But this tool does something else, it takes text ( I presume UTF-8 or something and turns it into a an mp3 file that can be played back on a computer or media player and that will sound like (but is not) the voice of an actor reading that text. The output isn't so much transformed input, as a new thing altogether that is in one vers specific aspect (the "text") identical to the input, and in every other aspect completely different. Oh, and they're also both digital files, but that wasn't the point. The point was that something doesn't get transformed, we need a different word for what happens when a .txt file becomes a .mp3. 

It could be media conversion: it *converts* representation of the text from one medium into another medium.

It _synthesizes_ speech from text.

It _renders_ text as sound.

it _encodes_ text in an audio format.

It performs *transduction*; the conversion of a signal from one kind of stimulus to another. Not an odeal termm ,it has for too  many specific meanings in different fields

It *generates* a new artifact, so it is a ***generative***  tool. It woiuld be nive to sticj  with terms that ise trans though.
