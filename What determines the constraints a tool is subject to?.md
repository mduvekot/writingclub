A tool can only operate on what the operator determines to be appropriate use: Using a screwdriver to open a paint can is not what the designer had in mind, but many people still use it for that purpose. Such operators use affordances as heuristics. Does the tool give the impression that it might work? 

What happens when we uses an LLM as a tool? It operates on language, which has syntaxical and grammatical constraints, but those don't guarantee that every utterance is meaningful. (Chomsky's *Colorless green ideas sleep furiously*). The user of the LLM doesn't impose any meaningful constraints. The LLM as a tool can say anything about anything in a way that looks correct, since it has syntax and grammar, but it is not materially bound (by physics etc.) It can say anything.
