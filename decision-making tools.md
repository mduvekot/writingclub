What kind of tools are these? The user is making a decision, based on assumptions. The outcome of the decision can be improved by  using better-quality assumptions that are more grounded in fact. Adding information (what has recently changed, what constraints should I observe, what is a successful example) improves outcomes.

All the mentioned tools, are data. Observed, recorded, structured quantitive statements about the world. Typically we think of this type of thing as a a material or a resource that a tool operates on, not as a tool itself, because the tool is the thing that transforms the resource/material  into something else. So it's reasonable to ask, what is the transformative function of data in this context? Who performs what function: What does that data do? And to what does it do it to? What are the operator, operation and operand in this context. As in carpenter(operator) uses screwdriver(tool) to to turn(operation) a screw(operand). 
 
If the operand is the mental model of the user. An investor(operator) uses data/a decision-support system(tool) to update(operator) their mental representation(operand) of the potential value of the outcomes of a decision. But in Zeno Vendler's model, the aspectual class of the verbs: **understand**, like **know** is a _state_, not an _activity_; something you can do to an operand.  

That data changes what the investor knows, so it's stative. Now what?  

The operand is not the state (knowing) but the process that causes change in state (updating knowledge). That is eventive, not stative. Verbs we might use here are notice, realize, update, and those have *outcomes*. 

Now, the tool is not the bare information, but rather its signalling function. "If something changes in a particular way, for example, when a value exceeds a threshold, notify me that I need to update my beliefs (priors)" Similar to how an alarm clock functions as a wake-up tool, The user sets a critical condition (at 7 AM) for the clock(tool) perform a function (operation: ring) that will cause a change(operand) in my state from sleeping to awake. 

In other words,  a decision-making tool is an attention-allocating device.
