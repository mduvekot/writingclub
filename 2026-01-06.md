“Californians can now use a government website to request that certain companies stop selling their personal information online. [The Drop website](https://privacy.ca.gov/drop/about-drop-and-the-delete-act/), which stands for Delete Request and Opt-Out Platform, launched on New Year’s Day as part of a state law aimed at enhancing data privacy.

The Drop tool sends a mass deletion request to 500 data brokers, the term for companies which collect and sell personal information gleaned from public records, phone data, online web browsing and a host of other activities. The information, which can be highly detailed and can be taken without user consent, is widely available for purchase on data brokers’ sites.” — [“New tool allows Californians to request data brokers delete personal details”, Nick Robins-Early, The Guardian](https://apple.news/AIzB8OwjNRZOjpNWro7KtmA)

Here is a "tool" that takes an identifier, an email address, phone number, Social Security Number (SSN) or Individual Taxpayer Identification Number (ITIN) and submits a request to data-brokers on behalf of the person with the ID.

The input to the tool is referent (for a person), the output of the tool is a binding compliance order to a corporate entity (company) to comply with a law (to remove or modify an entry in a database).
